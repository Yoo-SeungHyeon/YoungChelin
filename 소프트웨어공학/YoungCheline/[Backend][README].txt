Backend_README_FILE

* spring-boot-starter-data-jpa: 이 라이브러리는 Spring Framework에서 JPA(Java Persistence API)를 사용하여 데이터베이스를 다룰 수 있도록 도와줍니다. 데이터베이스와의 상호작용을 단순화하고 데이터를 객체로 매핑하는 데 도움이 됩니다.

* spring-boot-starter-mail: 메일 전송 기능을 지원해주는 라이브러리입니다. 이를 사용하여 이메일을 생성하고 전송하는 기능을 구현할 수 있습니다.

* spring-boot-starter-validation: 입력 데이터의 유효성을 검증하는 기능을 제공합니다. 사용자가 입력한 데이터의 형식이나 조건을 검사하고 유효성을 확인할 때 도움이 됩니다.

* spring-boot-starter-web: 웹 애플리케이션을 개발할 때 필요한 기본적인 웹 기능을 제공합니다. HTTP 요청과 응답을 처리하고 웹 애플리케이션을 구축하는 데 도움이 됩니다.

* spring-boot-starter-security: Spring Security를 사용하여 애플리케이션의 보안을 강화하는 기능을 제공합니다. 사용자 인증, 권한 부여 등 보안에 관련된 작업을 지원합니다.

* spring-cloud-starter-aws: AWS(Amazon Web Services)와 통합하여 클라우드 환경에서 애플리케이션을 개발할 수 있도록 도와주는 Spring Cloud의 AWS 모듈입니다.

* aws-java-sdk-s3: AWS의 S3(Simple Storage Service)에 접근하여 객체 스토리지를 사용할 수 있는 Java용 SDK입니다.

* javax.xml.bind:jaxb-api: XML 데이터와 Java 객체 간의 매핑을 지원하는 JAXB(Java Architecture for XML Binding) API입니다.

* org.json:json: JSON 데이터를 다루는 데 도움을 주는 라이브러리입니다.

* org.springframework.security:spring-security-test: Spring Security를 테스트하는 데 사용되는 라이브러리로, 보안 관련 기능의 테스트를 쉽게 작성할 수 있습니다.

* io.jsonwebtoken:jjwt-api, io.jsonwebtoken:jjwt-impl, io.jsonwebtoken:jjwt-jackson: JWT(JSON Web Tokens)를 생성하고 처리하는 데 사용되는 라이브러리들입니다.

* org.projectlombok:lombok: Java 코드의 반복적인 작업을 줄여주고 간편화하는 라이브러리입니다. 코드의 가독성을 높여주고 효율적인 개발을 돕습니다.

* com.mysql:mysql-connector-j: MySQL 데이터베이스와 연동하기 위한 JDBC 드라이버입니다.
